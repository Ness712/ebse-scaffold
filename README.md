# EBSE Guide

Guide universel de decisions techniques pour le developpement logiciel, base sur l'Evidence-Based Software Engineering (EBSE).

## Objectif

Fournir un guide ou chaque recommandation technique est sourcee, avec un niveau de confiance transparent, utilisable par n'importe quel developpeur ou entrepreneur.

## Methode

Adaptation de l'Evidence-Based Medicine (EBM) au genie logiciel, utilisant :

- **PICO** — formulation structuree des questions
- **Pyramide des preuves** — classement des sources par fiabilite
- **GRADE** — evaluation mecanique de la confiance
- **ISO/IEC 25010 x SWEBOK v4** — matrice de couverture exhaustive

Voir [methodology.md](methodology.md) pour la methode complete.
