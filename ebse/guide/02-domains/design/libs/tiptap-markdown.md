# Editeur Markdown

**[BONNE PRATIQUE]** **tiptap + react-markdown** | Score GRADE : 4/7

TipTap pour l'edition WYSIWYG du Markdown (avec tiptap-markdown pour serialisation), react-markdown pour l'affichage en lecture seule.

```bash
pnpm add @tiptap/react @tiptap/starter-kit tiptap-markdown react-markdown
```

| Alternative | Verdict |
|---|---|
| MDXEditor | Moins mature, moins d'extensions |
| react-simplemde-editor | Vieillissant, pas headless, bundle lourd |
