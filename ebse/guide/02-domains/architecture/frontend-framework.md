# Framework frontend

**[RECOMMANDE]** **React** | Score GRADE : 6/7

## Pourquoi React

2 agents independants arrivent a la meme conclusion : React.

| Critere | React | Vue.js | Svelte |
|---------|-------|--------|--------|
| Usage SO 2025 | **44.7%** (#1) | 17.6% | 7.2% |
| Satisfaction State of JS 2025 | 82.95% | 77.32% | **89.62%** |
| npm dl/semaine | **34M** | 4.3M | 1.4M |
| GitHub stars | **266k** | 235k | 91k |
| Component libraries | **MUI, shadcn/ui, Radix, Ant Design** | Vuetify, PrimeVue | Skeleton, limited |
| Hiring pool | **Le plus grand** | Moyen | Petit |

## Pourquoi pas Svelte (malgre la meilleure satisfaction)

Svelte a la satisfaction la plus haute (89.62%) mais :
- 7x moins d'adoption que React (7.2% vs 44.7%)
- Ecosysteme de composants beaucoup plus petit
- Hiring pool reduit — risque pour une equipe qui doit grandir
- La satisfaction elevee peut refleter un biais de selection (early adopters enthousiastes)

## Pourquoi pas Vue

Vue est un excellent #2 (courbe d'apprentissage plus douce). Mais :
- 2.5x moins d'adoption que React
- Ecosysteme de composants plus petit (pas de shadcn/ui equivalent mature)
- Hiring pool plus reduit

## Quand choisir autrement

- **Vue** si l'equipe le maitrise deja ou si la courbe d'apprentissage est prioritaire
- **Svelte** si l'equipe est petite, experimentee, et accepte un ecosysteme plus restreint
- **Angular** uniquement si enterprise avec besoin de framework ultra-opinionne

Sources : SO Survey 2025 49k devs (niv. 4), State of JS 2025 20k devs (niv. 4), JetBrains 2024 23k devs (niv. 4), npm trends (niv. 4), GitHub stars (niv. 4). Convergence de 5 sources independantes.
